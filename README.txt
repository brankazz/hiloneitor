Este programa crea hilos en Mastodon a partir de archivos de texto.
Para que funcione es necesario haber instalado 'toot' y haber iniciado sesión en la cuenta de Mastodon previamente (https://github.com/ihabunek/toot).
Se puede compilar el programa escribiendo "gcc -o hiloneitor hiloneitor.c" en cualquier máquina linux.
Para ejecutar el programa será suficiente indicarle el nombre del fichero de texto: "./hiloneitor texto.txt"
